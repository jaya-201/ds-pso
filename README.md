# ds-pso

Short repository for Particle Swarm Optimization (PSO) experiments.

**Project**
- `ds_pso.ipynb`: Jupyter notebook containing the PSO examples, experiments, and visualizations.

**Requirements**
- Python 3.8 or newer
- Jupyter Notebook or JupyterLab
- Common Python packages: `numpy`, `pandas`, `matplotlib`, `scipy` (install as needed)

**Quick Start**
1. Create and activate a virtual environment:

```powershell
python -m venv .venv; .\.venv\Scripts\Activate.ps1
```

2. Install common packages (adjust as needed):

```powershell
pip install --upgrade pip
pip install jupyter numpy pandas matplotlib scipy
```

3. Launch Jupyter and open the notebook:

```powershell
jupyter notebook ds_pso.ipynb
```
